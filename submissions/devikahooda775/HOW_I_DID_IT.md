# HOW_I_DID_IT

First, I cloned the LPI Developer Kit repository and installed dependencies using npm install. Then I built the project with npm run build and verified the tools using npm run test-client.

After that, I installed Ollama and ran a local LLM. I used it to generate output related to the SMILE methodology and saved that as part of my submission.

While doing this, I learned how the LPI sandbox works, how MCP tools can be tested locally, and how a local LLM can be used alongside them. I also faced some Git and PR issues and learned how to manage branches, restore files, and keep submissions clean.
